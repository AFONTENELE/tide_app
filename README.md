# mare_app

Este projeto é um aplicativo desenvolvido para facilitar o monitoramento e análise de dados relacionados às marés. Ele oferece uma interface intuitiva para visualização de informações, gráficos e alertas, auxiliando usuários na tomada de decisões baseadas em dados marítimos. O objetivo principal é fornecer uma ferramenta eficiente e acessível para pesquisadores, pescadores e entusiastas do ambiente costeiro.

## Funcionalidades

- Consulta de dados de maré em tempo real
- Visualização gráfica dos níveis de maré
- Alertas personalizados
- Histórico de dados
